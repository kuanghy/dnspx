# 公共 DNS 服务介绍

## 国内

### 114 DNS

由南京信风牵头，联合电信运营商采用 BGP AnyCast 技术多点部署的公共 DNS 平台。提供三种模式：


| 模式      | 首选                | 备选                | 说明      |
| ------- | ----------------- | ----------------- | ------- |
| 纯净（无劫持） | `114.114.114.114` | `114.114.115.115` | 默认模式    |
| 拦截钓鱼/病毒 | `114.114.114.119` | `114.114.115.119` | 增强安全    |
| 拦截色情内容  | `114.114.114.110` | `114.114.115.110` | 家庭/校园防护 |


> 不支持 DoT / DoH。

### 腾讯 DNSPod（Public DNS+）

DNSPod 提供的公共 DNS 服务（DNSPod 现为腾讯旗下），全球范围内稳定性和连通性表现良好。


| 协议   | 地址                                 |
| ---- | ---------------------------------- |
| IPv4 | `119.29.29.29` / `182.254.116.116` |
| IPv6 | `2402:4e00::` / `2402:4e00:1::`    |
| DoT  | `dot.pub`（端口 853）                  |
| DoH  | `https://doh.pub/dns-query`        |

### OneDNS

| 协议   | 首选                                 | 备选                       |
| ---- | ---------------------------------- | ------------------------ |
| IPv4 | `52.80.66.66`                      | `117.50.22.22`           |
| IPv6 | `2400:7fc0:849e:200::4`            | `2404:c2c0:85d8:901::4`  |
| DoT  | `dot.onedns.net`（端口 853）           |                          |
| DoH  | `https://doh.onedns.net/dns-query` |                          |

**家庭版**（额外拦截不良内容）

| 协议   | 首选                                      | 备选                       |

| DoT  | `dot-home.onedns.net`（端口 853）           |                          |
| DoH  | `https://doh-home.onedns.net/dns-query` |                          |

| 协议   | 地址                                             |
| ---- | ---------------------------------------------- |

### Quad9

由 IBM、PCH 和全球网络联盟（GCA）联合发起的非营利性、主打隐私保护和安全的公共 DNS 服务。内置恶意软件和钓鱼网站拦截功能，不记录用户 IP 地址，符合最严格的隐私标准。

| 协议   | 地址                                 |
| ---- | ------------------------------------- |
| IPv4 | `9.9.9.9` / `149.112.112.112`         |
| IPv6 | `2620:fe::fe` / `2620:fe::9`          |
| DoT  | `dns.quad9.net`（端口 853）            |
| DoH  | `https://dns.quad9.net/dns-query`     |

### Level 3 DNS (Lumen)

Level 3 DNS 是互联网历史上最早的公共 DNS 服务之一，由 Level 3 Communications（现 Lumen Technologies）运营。其标志性地址 4.2.2.2 在 Google DNS（8.8.8.8）出现之前，是网络工程师最常用的公共 DNS 和网络连通性测试地址。

| 协议  | 首选                  | 备选                     |
| ---- | --------------------- | ----------------------- |
| IPv4 | `4.2.2.1` / `4.2.2.2` | `4.2.2.3` ~ `4.2.2.6`   |

> 不支持 DoT / DoH。
